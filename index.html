日本社会への影響

封鎖から8ヶ月間を想定した社会インフラ・物価・産業への連鎖的影響の時系列分析

時系列フェーズ別シナリオ

封鎖直後から8ヶ月目まで、5つのフェーズで何が起きるかを示す。各フェーズをクリックして展開。

```
PHASE 01
初動ショック期
1〜2週目
+
  • 原油先物が即座に急騰（+30〜50%）、市場が混乱
  • ガソリンスタンドで買い占め行動が散発的に発生
  • 政府・資源エネルギー庁が緊急会議、備蓄放出を検討開始
  • 航空会社が燃油サーチャージの緊急引き上げを発表
  • 電力・ガス会社が「料金見直しの可能性」を公表
  • 報道各社が特番、消費者心理が急速に悪化
PHASE 02
備蓄対応・第一波インフレ
2〜6週目
+
  • IEA協調備蓄放出が発動（日本は約145日分の戦略備蓄）
  • ガソリン価格が+40〜60円/L（200円台後半〜300円前後）に
  • 電気料金・都市ガス料金の緊急値上げ改定（+20〜35%）
  • 食品・日用品メーカーが相次いで値上げを発表（運送費増大）
  • プラスチック原料不足の懸念から包装材が値上がり
  • 農業用肥料・農薬の高騰で農家が生産縮小を検討
  • 政府が「節電要請」「ガソリン補助金強化」を発動
PHASE 03
物資不足・産業停滞期
2〜3ヶ月目
+
  • 石油化学製品（プラスチック・合成繊維・洗剤原料）の供給不安が顕在化
  • 自動車・電子機器メーカーが減産を検討（部品・素材コスト増）
  • トラック輸送コスト急増→流通に深刻な影響、宅配料金値上げ
  • 地方のガソリンスタンド閉鎖・統廃合が加速（採算悪化）
  • 農産物・水産物の価格が顕著に上昇（燃料・肥料・輸送コスト）
  • 政府が「省エネ強化令」を発令、大型施設の夜間照明規制など
  • 株式市場が下落基調、円安が進行（輸入物価のさらなる押し上げ）
PHASE 04
社会機能への本格的圧力
3〜5ヶ月目
+
  • 戦略備蓄の残量が意識され始め、節約・割当制の議論が浮上
  • 製造業の一部で操業停止・短縮（自動車・化学・鉄鋼など）
  • 病院・救急施設での燃料調達コスト上昇、医療費への転嫁検討
  • 公共交通の運賃値上げ（バス・タクシー先行、鉄道も検討）
  • 農作物の収穫・出荷に支障（農業用機械燃料不足）
  • 中小企業の倒産件数が増加（エネルギーコスト増が直撃）
  • 消費者物価指数（CPI）が前年比+8〜12%台に到達
  • 外食・宿泊業が相次ぎ値上げ、消費者の外出・消費を抑制
PHASE 05
危機的局面・構造的変容
5〜8ヶ月目
+
  • 石油割当制度・配給的な仕組みの導入が現実的な政策課題に
  • 電力逼迫→計画停電の可能性（夏または冬と重なった場合は深刻）
  • 輸出産業の競争力低下、貿易収支が大幅悪化、GDP押し下げ
  • 物価高騰による実質賃金の大幅低下、消費の急速な冷え込み
  • 社会インフラ（水道・ごみ収集・医療搬送）への燃料優先配給
  • 食料品の入手困難が一部地域・品目で発生
  • 政府が緊急経済対策・補正予算を編成（数十兆円規模）
  • 中東情勢に応じた外交・安全保障の再定義が急務に
  • 再生可能エネルギー・原子力の緊急拡大議論が加速
```

社会基盤分野別リスク評価

電力・物流・農業・医療など各分野が受ける打撃の深刻度と具体的な影響内容。

```
電力供給
深刻
LNG・石油火力が発電量の約40%を担う。長期化すれば電力需給が逼迫。夏冬のピーク時に計画停電のリスク。再エネ・原子力への転換は数ヶ月では限界。
交通・物流
極めて深刻
トラック物流のほぼ全量が軽油依存。運送コスト急増が全品目に転嫁。地方への物資供給が先に細る。航空貨物・フェリーも打撃。
農業・食料生産
深刻
農業機械の燃料・農薬・化学肥料がすべて石油由来。生産コスト急増で収穫量減少の可能性。漁業は燃油費が経営を直撃し出漁停止も。食料自給率の低さが露呈。
製造業・産業
深刻
石油化学・鉄鋼・セメント・自動車が連鎖的打撃。原料調達コスト増→減産・操業停止。輸出競争力低下で貿易赤字が急拡大。雇用悪化リスク。
医療・救急体制
中〜深刻
救急車・ドクターヘリの燃料確保は最優先。病院の光熱費急増で経営悪化。医療用プラスチック・医薬品原料の調達リスク。透析・ICUなど電力多消費の医療行為に停電リスク。
上下水道・廃棄物
中程度
浄水場・ポンプ場は電力依存。停電が長引けば水道供給に支障。ごみ収集車は燃料高騰の影響を受ける。塩素・薬品調達コストも上昇。
通信・IT基盤
低〜中
データセンター・基地局は電力依存だが非常用発電を装備。停電が長期化すれば通信断リスク。半導体・電子部品製造は素材コスト増。直接的燃料依存は低い。
金融・財政
深刻
貿易赤字の急拡大→円安加速→輸入物価のさらなる上昇という悪循環。財政出動（補助金・給付）による国債増発。企業倒産増加で不良債権増大リスク。
```
本資料はシミュレーションであり、実際の政策・市場の動向を保証するものではありません。
過去の石油危機（1973年・1979年）および各種エネルギー統計をもとに推計。
